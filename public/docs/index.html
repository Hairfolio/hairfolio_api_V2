REST API v.1.0

Brands

These endpoints belong to the brands resource.
Resource Description
GET /brands Getting available brands
GET /brands/:id Getting the brand details
Categories

These endpoints belong to the categories resource.
Resource Description
GET /categories/ Getting the available categories
Certificates

These endpoints belong to the certificates resource.
Resource Description
GET /certificates/ Getting the available certificates
GET /certificates/:id Getting the information on a certificate
Comments

These endpoints belong to the comments resource.
Resource Description
GET /posts/:post_id/comments Getting the comments on a post
POST /posts/:post_id/comments Creating a comment on a post
PUT /posts/:post_id/comments/:id Updating a comment on a post
DELETE /posts/:post_id/comments/:id Deleting a comment from a post
Contacts

These endpoints belong to the contacts resource.
Resource Description
GET /contacts Displaying the logged in user's contacts ( Black book )
GET /contacts/:id Displaying the contact information
POST /contacts Creating a contact
PUT /contacts/:id Updating a contact
DELETE /contacts/:id Deleting the contact from the black book
Conversations

These endpoints belong to the conversations resource.
Resource Description
GET /conversations Getting the conversations the current user is a participant of
POST /conversations Starting a conversation with users
POST /conversations/:id Updating a conversation to add or delete users
DELETE /conversations/:id Deleting a conversation and it's messages
Degrees

These endpoints belong to the degrees resource.
Resource Description
GET /degrees Getting available degrees
GET /degrees/:id Getting a single degree
Experiences

These endpoints belong to the experiences resource.
Resource Description
GET /experiences Getting available experiences

These endpoints belong to the home resource.
