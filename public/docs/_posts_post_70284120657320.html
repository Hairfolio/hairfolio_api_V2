POST /posts

Description

Creating a post as the logged in user

Headers

Accept

application/vnd.hairfolio.v1

Content-Type

application/json

Parameters

post[description] required, type: string

This is the body of the post

post[labels_attributes][0][tag_id] required, type: integer

This is the id of the tag to associate the post with

post[labels_attributes][0][position_top] required, type: integer

This is the position from the top to place the tag

post[labels_attributes][0][position_left] required, type: integer

This is the position from the left to place the tag

post[photos_attributes][asset_url] required, type: string

This is the service of the formula to add

post[photos_attributes][0][formulas_attributes][0][service_id] required, type: integer

This is the service of the formula to add

post[photos_attributes][0][formulas_attributes][0][weight] required, type: integer

This is the weight of the formula to add

post[photos_attributes][0][formulas_attributes][0][volume] required, type: integer

This is the volume of the formula to add

post[photos_attributes][0][formulas_attributes][0][time] required, type: integer

This is the time of the formula to add

post[photos_attributes][0][formulas_attributes][0][treatments_attributes][weight] required, type: integer

This is the weight of the formula treatment

post[photos_attributes][0][formulas_attributes][0][treatments_attributes][color_id] required, type: integer

This is the color id of the formula treatment
